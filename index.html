Te damos la bienvenida al OVI Construcción de Bases de Datos Relacionales. En este espacio aprenderás los temas y habilidades mínimas para la construcción de un modelo lógico de datos, aplicado a las bases de datos relacionales

Con esta información analizaras un problema, identificaras las entidades, indagaras de cuáles son las posibles relaciones que existen entre entidades y cuales se pueden ajustar a las necesidades de tu proyecto o trabajo, generaras el modelo lógico de datos con sus respectivos atributos.
