Para poder realizar un diagrama de entidad relación necesitamos comprender cuales son las distintas relaciones que existen y su cardinalidad.

Relación 1-1

Este tipo de se usa cuan dos entidades se relacionan con un solo elemento y su cardinalidad se representa 1-1. ejemplo: 1 ser humano solo puede tener 1 solo corazón.

Relacion 1-N Este tipo de relación se usa cuando un elemento de una entidad puede ser asignado a varios elementos de la segunda entidad. Se representa 1-N. ejemplo.

1 edificio puede tener varios departamentos. Relación N-N Este tipo de relación se da en algunos casos, pero regularmente se finaliza creando una entidad intermedia, ya que este tipo de relación dentro de las bases de datos relacionales no puede quedar configurado con este tipo de cardinalidad.
