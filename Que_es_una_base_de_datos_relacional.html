En esta breve descripción hablaremos de que es una base de datos. Una base de datos es un sitio donde se puede almacenar mucha información, el nombre relacional esta dado a este tipo de base de datos por que la información contenida presenta una relación entre sus entidades. La información se guarda en tablas compuestas por columnas y filas, en cada columna se clasifica la información que es el tipo de dato que lo identifica, las filas serán cada registro de dato. En el siguiente ejemplo veremos cómo está constituida una tabla, en las columnas podemos ver los títulos (Identificación, Nombre y apellido) estos títulos representan los atributos dentro de la tabla, los datos ingresados están identificados por filas, cada fila representa un dato diferente en la tabla.
